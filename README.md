# JivaHealth-Assignment

## Tech Stack
- React 18 + TypeScript
- Vite
- Tailwind CSS
- Zustand
- React Router v6

## Setup
```bash
npm install
npm run dev
```

## Features
- User list with search and status filtering
- User detail with 4 tabs: Overview, Orders & Bookings, Payments, Family Members
- Add/Edit users and family members
- Upgrade users to Prime
- Toggle Active/Inactive status
- Order status management
- Address management

## Notes
- All data is mock/in-memory (no backend)
- State persists within session via Zustand
